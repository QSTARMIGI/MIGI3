# MIGI3

**MIGI3** is an experimental workspace in the QSTARMIGI portfolio.

## Role

Use this repository for bounded prototypes that may inform MIGI Core but are not yet part of the canonical implementation.

Every experiment should state:

- the problem it explores;
- upstream code, assets, models, or datasets used;
- license and provenance status;
- whether outputs are original, derived, or simulated;
- what result would justify migration into `QSTARMIGI/MIGI`.

## Migration Rule

An experiment should move into MIGI Core only after it has:

```text
clear scope
versioned schema or interface
provenance record
policy implications
basic validation
security review appropriate to risk
documentation distinguishing implemented from planned behavior
```

## Status

`experimental` — not a production release and not the canonical source of MIGI authority, receipt, or policy behavior.

See [QSTARMIGI/MIGI](https://github.com/QSTARMIGI/MIGI) for the canonical architecture and MIRROR SEED MVP backlog.
